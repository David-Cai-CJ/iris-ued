Module contents¶

Allows easy loading of pixmaps used in UI elements. Provides support for frozen environments as well.

iris.gui.pyqtgraph.pixmaps.getPixmap(name)[source]¶

Return a QPixmap corresponding to the image file with the given name. (eg. getPixmap(‘auto’) loads pyqtgraph/pixmaps/auto.png)
